W IDE

W IDE (or Web IDE) is a simple Web Development IDE which can be used to execute HTML, CSS, JavaScripts and get the results as soon as it is typed :)

Useful for people who are new to Web Development and want to brush up their skills in a friendly environment.

Being able to simultaneously write codes and see the results is a very intuitive and an easy way to learn to code.
